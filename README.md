
# Lista de compras

## Objetivo

A razão desse projeto originou-se de uma necessidade de um familiar de usar um programa capaz de mostrar o preço total de uma compra com base nos itens que foram registrados em tempo real, no entanto, a capacidade de armazenamento deste celular (Android) estava limitada e instalar um aplicativo que faça isso significava sacrificar um pouco de armazenamento para alguma funções que o aplicatovo oferece mas o usuário não usará durante uma compra.

Sendo assim, criei uma (e somente uma) página em HTML que pudesse fazer, não tudo, mas o necessário que o usuário pudesse controlar o quanto ele vai gastar em uma compra. Uma solução leve que é rodada em qualquer navegador instalado no Android/IOS.

## Dificuldades

Apesar do programa em si ser bem objetivo, é necessário certa experiência do usuário em pelo menos abrir esse tipo de arquivo. Claro que esse conhecimento é bem básico, ainda mais considerando que ele foi feito para ser enviado via Whatsapp, por exemplo, mas nem todas as pessoas sabem que um arquivo em HTML precisa ser aberto pelo navegador.

A  única maneira de resolver isso, se não alterar o título para "ABRA COM O CHROME.html" ou algo parecido, foi alertar a pessoa antes que ela abra o programa.
